# Grand Unifying Theory Potential Framework (v7.21)

A MATLAB implementation of a variational efficiency-based Grand Unifying Theory that unifies all fundamental forces from a single potential.

## Key Features
- Resolves Hubble tension without extra physics
- Realistic boson and fermion masses from SU(5) breaking
- Proton decay branching ratios
- Cosmic string tension
- LISA gravitational-wave prediction
- Vacuum stability, unitarity, gauge unification, DM relic, non-Gaussianity, BBN, FCNC, seesaw neutrinos
- Monte Carlo uncertainty bands

## How to Run
1. Open MATLAB
2. Run `gut_framework.m`
3. All plots and results appear automatically
